## 👋 Welcome

If you've ever thought *"I want to get into Cloud or AI, but I have no idea where to start"* — this is for you.

Over **5 days and 10 hours of live teaching**, we go from *"what even is the cloud?"* to watching real AWS services work together to build something tangible. No prior experience required. No jargon for the sake of jargon. Just a clear, friendly path into the skills employers are hiring for right now.

This repository is your **companion resource** for the cohort — slides, code samples, demo files, and links you can keep forever.

## 🎯 Who This Is For

✅ Students & recent graduates exploring tech careers
✅ Professionals from non-IT backgrounds curious about cloud
✅ Early-career IT folks wanting to specialize in AWS
✅ Anyone who's been *"meaning to learn AWS"* for a while

## 📚 What You'll Learn — Day by Day

| Day | Topic | What we cover |
|:---:|:------|:--------------|
| **1** | ☁️ **Cloud & AWS Basics + IAM** | What cloud computing actually is, how AWS is organized, and getting hands-on with Identity & Access Management |
| **2** | 🗄️ **Amazon S3 & CloudFront** | Object storage and global content delivery — the foundation of almost everything on AWS |
| **3** | ⚙️ **Compute Services** | EC2, Lambda, and ECS — the core building blocks for running applications |
| **4** | 🤖 **A Gentle Intro to AWS AI** | Meet Bedrock, Rekognition, Polly & Textract — the services powering modern AI apps |
| **5** | ⭐ **Live Build Demo Day** | See how Lambda, Textract, Bedrock, S3 & CloudFront chain together to build something real |

## 🚀 Getting Started

Everything you need to follow along:

1. **An AWS account** — the [AWS Free Tier](https://aws.amazon.com/free/) is plenty for everything we do
2. **A web browser** — most of what we cover happens in the AWS Console
3. **Curiosity** — by far the most important requirement 🙂

> ⚠️ **A note on costs:** We stay within the AWS Free Tier wherever possible. Always remember to clean up resources after experimenting (we'll show you how) so you don't get surprise charges.
